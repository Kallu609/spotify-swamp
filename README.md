# spotify-swamp

Finds records from Spotify playlists from supported record shops.

Currently supported:

- [Swamp Music](https://www.swampmusic.com/)

## Usage

1. Add this to your `.env`

```
SPOTIFY_API_ID="your spotify api id"
SPOTIFY_API_SECRET="your spotify api secret"
SPOTIFY_PLAYLIST_ID="the playlist you want to crawl"
```

2. Run `npm start`
3. Profit
